# QR Reader App

A React Native QR Reader application built using Expo Snack.

## Features

- User Login Screen
- User Signup Screen
- Home Screen
- Detail Screen
- API Integration
- Local Storage Support
- Settings Screen
- Notifications Screen
- Error Handling Screens

## Technologies Used

- React Native
- Expo Snack
- JavaScript
